# Pokémon Deck Checklist

A single-file printable checklist (`index.html`) for one deck:
. No server or dependencies — just open it.

## What it does

- Shows the full 60-card deck as a printable checklist with a check box for every entry.
- Groups cards into sections:
  - **Pokémon** — by evolution line (Greninja line, Dragapult line) plus standalone Pokémon.
  - **Trainers** — Supporters, Items, Pokémon Tools, Stadiums.
  - **Energy** — Basic and Special.
- Each row shows a check box, the quantity, the card name, and the set code + number.
- The header lists the section counts (21 Pokémon · 29 Trainers · 10 Energy) and the 60-card total.

## Printing

Click **Print** (top-right). The button is hidden on the printed page, and the list lays
out in two columns to fit cleanly on paper.

## Editing

It's a static page — to change cards, edit the list items directly in `index.html`.
